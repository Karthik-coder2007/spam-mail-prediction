# spam-mail-prediction

Spam Mail Prediction using Logistic Regression:
This project implements a Spam Email Classifier using Logistic Regression, a popular machine learning algorithm for binary classification. The model predicts whether an email is Spam or Not Spam (Ham) based on the email content.

Machine Learning Approach
Algorithm Used: Logistic Regression
Type: Supervised Learning
Problem Type: Binary Classification (Spam / Ham)

Technologies & Libraries Used

Python 

NumPy

Pandas

Scikit-learn

NLTK (for text preprocessing)

Jupyter Notebook / Google Colab
